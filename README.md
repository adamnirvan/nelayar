# Nelayar 🌊
**Integrated Web GIS Platform with Oceanographic Forecasting Data for Precision Pelagic Fishing**

## 📖 About The Project
Nelayar is a Geographic Information System (GIS) based application designed to improve the efficiency and catch rates of fishermen. The system processes and visualizes satellite oceanographic data to provide precise recommendations for pelagic fishing zones. Built with a mobile-first approach, it features a fully responsive interface, ensuring fishermen can easily and comfortably access critical information directly from their smartphones while at sea.

## ✨ Key Features

### 1. Interactive Marine GIS Mapping
Interactive mapping of marine waters featuring markers for high-potential fishing zones across Indonesia.

### 2. Oceanographic Data Pipeline Integration (CMEMS)
Real-time visualization of marine environmental data layers directly sourced from Copernicus satellites.

### 3. 10-Day Forecasting
An interactive time-slider system to predict changes in sea conditions and fish movement patterns up to 10 days ahead.

### 4. Comprehensive Area Details
An integrated information panel providing in-depth data for any coordinate point selected by the user.
> `<img width="2880" height="1556" alt="Nelayar-06-12-2026_07_15_PM" src="https://github.com/user-attachments/assets/ea29bd36-8b38-4191-97e4-d69b69be688f" /><img width="2880" height="1556" alt="Nelayar-06-12-2026_07_15_PM (1)" src="https://github.com/user-attachments/assets/235d547c-242d-42d4-9320-c3f79148fbfb" />
`
* **Oceanographic Data (CMEMS):** Sea Surface Temperature (SST) and Chlorophyll-a (Chl-a) levels.
* **Weather Conditions (Open-Meteo):** Real-time weather reports, wind speed, and ocean wave height.
* **Logistics Calculation:** Voyage distance estimation and projected fuel costs.
* **AI/Catch Logic:** Probability and prediction of pelagic fish species in the selected area.

### 5. Real Sea Routing
Precision maritime navigation system utilizing Searoutes API integration to determine the safest and most fuel-efficient vessel routes.
> `<img width="2880" height="1556" alt="Nelayar-06-12-2026_07_15_PM (1)" src="https://github.com/user-attachments/assets/6ad4129b-3a8e-4b7c-95b7-5e3cb693a9f5" />`

### 6. Offline Mode (PWA)
Powered by Progressive Web App (PWA) technology and advanced caching, the application remains functional offline. Fishermen can access base maps, saved routes, and the latest forecast data without an internet connection in the middle of the ocean.

### 7. Fish Market Price Monitor
Real-time tracking of marine commodity price fluctuations, which can be filtered by fish species and local fish auction locations.
